React is FE libraries
Used for handling View
Allows to create reusable UI components


JSX - JavaScript Syntax Extension.
Babel - allow us to use next generation JS.

JavaScript Expressions can be used inside of JSX using curly braces.
Like { 1 + 1 }

You can't use other expressions like if..else loop inside JSX but can be done using something else.

React recommend inline style using camelCasing syntax. It will also automatically append 'px' after the number value.
